# 项目简介
本项目基于Python3.7 Scrapy爬虫框架，针对epubw.com网站的电子书进行爬取，
并且针对爬取到的百度网盘链接进行selenium自动化保存，仅限个人研究开发，切勿用于商业用途！

# 技术栈
Python 3.7
Scrapy
Selenium
MySQL
Redis
